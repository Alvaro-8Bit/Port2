Função Exponencial:
A0 = 1;
Am * An = Am + n;
Am / An = Am - n;
(Am)n = Am * n;
(A * B)m = Am * Bm;
(A / B)m = Am / Bm;
A-n = 1 / An , para A ≠ 0 e n > 0;
(A / B)-n = (B / A)n , para n > 0;
Am / n = n√Am;

Função Logarítmca:
Logm(A * B) = Logm(A) + Logm(B);
Logm(A / B) = Logm(A) - Logm(B);
Logm(AB) = B * Logm(A);
LogB(A) = LogC(A) / LogC(B);
Logm(1) = 0;
