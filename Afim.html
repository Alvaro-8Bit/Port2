FUNÇÃO AFIM

 Função Quadrática Função Exponencial Função Logarítmica Conteúdo Extra





   As funções afim são representadas pela equação Ax + B. São representadas por retas na projeção gráfica, já que os números são apenas acrescidos constantemente e não possuem variação no aumento do valor. O número “B” serve para nos mostrar onde o ponto que cruza o eixo “y” vai estar, já que o eixo “x” é cortado quando “x” = 0, e se “x” = 0, então F(0) = A * 0 + B = 0 + B = B. O número “A” serve para indicar se a representação gráfica é crescente ou decrescente, já que se “A” for positivo o “F(x)” vai crescer gradativamente e se for negativo, “F(0)” decresce. Para você achar o “A” de uma função determinada apenas a partir do resultado de duas funções já resolvidas, basta usar a equação “A = (F(x2) - F(x1)) / (x2 - x1)”, em que “x2” é maior que “x1”. Para encontrar o ponto de contato entre duas funções afim, basta apenas igualá-las.

Domínio da função afim: Qualquer valor do conjunto dos números reais.
Imagem da função afim: Qualquer valor do conjunto dos números reais.

Aproximação de um gráfico da função afim F(x) = x + 1.
